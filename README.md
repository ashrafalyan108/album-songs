# m9
Proyecto M9 ASIX (PHP,Bootstrap)

Se trata de un CRUD con datos de canciones y artistas que interactua con mysql y tiene un frontend creado a partir de plantillas de Bootstrap. 

Desde los diferentes roles se pueden administrar usuarios (insert en BD), añadir y borrar, así como modificar los contenidos de la base de datos de canciones.

Es para el módulo M9 del CFGS de ASIX/ASIR.

Autores: Alyan Ali(@malyalin), Carlos Lopez (@clopez7), creador de la plantilla original de bootstrap (@mdo)
